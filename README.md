# Сайт it-компании softintermob  

## Установка и запуск проекта

### Клонирование репозитория

```bash
git clone https://github.com/gineff/soft-inter-mob
cd soft-inter-mob
```

### Установка зависимостей

```bash
npm install
```

### Запуск приложения

```bash
npm start
```
